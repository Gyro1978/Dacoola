WordFinder App: Harnessing Generative AI on AWS for Aphasia Communication

Developed by researchers from the Queensland Aphasia Research Centre (QARC) and AWS, the WordFinder app leverages AI for aphasia to assist individuals struggling with communication disabilities. The mobile app, born from the 2023 AWS Hack For Purpose event, uses generative AI to generate word lists from user-selected images, helping bridge language gaps for over 140,000 Australians affected by aphasia—a number expected to double by 2050.

The app’s workflow begins with image recognition via Amazon Rekognition, which identifies objects in uploaded photos. AWS Lambda and Amazon Bedrock then expand these labels into semantically related words, offering users alternative ways to express themselves. For example, selecting an image of a koala might yield terms like "marsupial" or "eucalyptus," aiding those who struggle with word retrieval. Built with React Native and Expo, WordFinder ensures cross-platform accessibility while AWS Amplify handles authentication and storage.

Significance

By integrating generative AI with real-world user insights—including input from Scott Harding, a person living with aphasia—WordFinder exemplifies how technology can empower marginalized communities. Its scalable AWS architecture, featuring API Gateway and lifecycle-managed S3 storage, ensures efficient, secure operation while reducing barriers to communication.
