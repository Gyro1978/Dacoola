Amazon Bedrock Model Distillation: Boost Function Calling Accuracy While Reducing Cost and Latency

Amazon Bedrock Model Distillation is now generally available, offering organizations a way to enhance generative AI performance while lowering operational costs and latency. This technique transfers knowledge from larger "teacher" foundation models (FMs) to smaller, more efficient "student" models, enabling specialized task execution without sacrificing accuracy. A key application is agent function calling, where distilled models achieve comparable precision to their larger counterparts while delivering faster responses and reduced expenses.

The process leverages advanced data augmentation techniques, particularly with Meta’s Llama model family, to refine function-calling capabilities. Smaller models trained through distillation can accurately determine when and how to invoke external tools, APIs, or databases—a critical requirement for modern AI applications. Performance evaluations show significant improvements, with distilled models achieving up to a 63.8% accuracy boost in simple function-calling tasks and a 26.6% gain in complex scenarios compared to base models.

Beyond accuracy, the economic benefits are compelling. Distilled models like Llama 3.2 3B offer a 72% reduction in latency and a 140% improvement in output speed compared to larger models like Llama 3.1 405B. Additionally, operational costs drop substantially, making sophisticated AI agents viable for broader enterprise deployment. With expanded model support and enhanced training transparency, Amazon Bedrock Model Distillation provides a scalable solution for businesses balancing performance and efficiency in AI-driven workflows.
