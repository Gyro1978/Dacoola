JetBrains Releases Mellum, an Open AI Coding Model

JetBrains has unveiled Mellum, its first open AI model designed for code generation. The model, previously exclusive to JetBrains' software development suites, is now publicly accessible on Hugging Face.

Trained on over 4 trillion tokens, Mellum aims to enhance developer productivity by automating coding tasks. The release marks JetBrains' entry into the growing open AI model space.
